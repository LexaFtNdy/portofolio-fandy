Featured Project

Web Profile.

Personal Profile & CMS Platform

A white-label web profile and CMS with dynamic OG meta tags, rich text rendering, and a robust admin dashboard for complete content management.

The Vision

The goal was to create a production-ready, white-label CMS platform designed for personal web profiles that is highly customizable and easy to maintain. I wanted to build a stable, polished solution that can be seamlessly scaled for future clients.

The Features

I implemented dynamic OG meta tags to ensure social sharing works perfectly via social media. The platform handles rich text rendering accurately and delivers content blazingly fast. All of this is manageable through a secure admin dashboard.

The Deployment

The entire application is containerized with Docker and deployed on a resource-constrained VPS. I configured Caddy as a reverse-proxy to manage SSL and routing elegantly.

Tech Stack

Frontend & CMS

Built with Vue.js and Tailwind CSS for a modern, responsive, and seamless user experience. Includes rich-text rendering integration.

Infrastructure

Deployed via Docker with Caddy handling reverse-proxy and automatic HTTPS, optimized for limited-resource VPS.
