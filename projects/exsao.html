Featured Project

Exsao.

Computer-Based Testing Platform

A multi-tenant SaaS examination system built to handle the chaos of hundreds of students clicking "Submit" at the exact same second.

The Origin

Exsao started as a web-only exam platform commissioned by KKMI Kemenag Surabaya (Ministry of Religious Affairs). No desktop app, no mobile client — just a straightforward browser-based CBT system for their school network. I built it, delivered it, and it worked. But I saw a bigger opportunity — why build this for one institution when every school in the region faces the same problem?

The Problem

Schools across East Java needed a unified, reliable digital exam system. The catch? 600+ students taking exams simultaneously, auto-saving answers every few seconds, with zero tolerance for data loss. Traditional CBT solutions either crashed under load or leaked exam content. Neither was acceptable.

The Evolution

I re-architected the entire codebase from a single-school web app into a multi-tenant SaaS platform. Then I built what the original never had — a Tauri-based secure desktop browser to lock down the testing environment, a Capacitor-based mobile app for flexible device support, and a centralized super admin dashboard to orchestrate everything. What started as one school's web-based exam tool now serves 11+ institutions as a full product ecosystem.

The Ecosystem

CBT Server

Multi-tenant Laravel backend with database-per-tenant isolation. Handles concurrent exam sessions with row-level locking, bulk upserts for auto-save, and HMAC-authenticated cross-service communication.

LaravelVue.jsInertiaMySQL

Secure Desktop

Tauri-based locked browser that blocks Alt+Tab, screen capture, and unauthorized processes. Features hardware-level device fingerprinting and real-time violation tracking sent directly to the exam proctor.

TauriReactRustTypeScript

Mobile App

Capacitor + Vue mobile exam client with offline-first architecture. Local persistence ensures no answer is ever lost — even in areas with unstable connectivity. Syncs seamlessly when back online.

CapacitorVue.jsTypeScriptIonic

Super Admin

Centralized dashboard for managing the entire school network. Orchestrates collaborative multi-institution exam events, aggregates leaderboard data in real-time, and generates cross-school analytical reports.
